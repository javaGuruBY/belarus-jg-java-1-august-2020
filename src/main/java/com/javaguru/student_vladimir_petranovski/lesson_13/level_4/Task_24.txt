Знакомство с классами ошибок.

Найдите в документации Java (javadoc) для чего испольхуются
следующие ошибки:
- ArrayIndexOutOfBoundsException -- Брошено, чтобы указать, что к массиву обращались с недопустимым индексом. Индекс либо отрицателен, либо больше или равен размеру массива.

- IllegalArgumentException -- Брошено, чтобы указать, что методу был передан недопустимый или несоответствующий аргумент.

- IOException -- Сигналы о том, что произошло какое-то исключение ввода-вывода. Этот класс является общим классом исключений, вызванных неудачными или прерванными операциями ввода-вывода.

- StackOverflowError -- Выдается, когда происходит переполнение стека из-за слишком глубокой рекурсии приложения.

- OutOfMemoryError -- Вызывается, когда виртуальная машина Java не может выделить объект, потому что ему не хватает памяти, и сборщик мусора не может сделать больше памяти. OutOfMemoryErrorобъекты могут быть созданы виртуальной машиной, как если бы подавление было отключено и / или трассировка стека не была доступна для записи .

- NumberFormatException -- Брошено, чтобы указать, что приложение попыталось преобразовать строку в один из числовых типов, но строка не имеет соответствующего формата.
