User - дата класс
------------- UI
UserMenu - UI. Console. Работа с пользователем.

------------- Business Logic
UserService - бизнес логика. Проверять корректность User. Вызывать репозиторий или другие классы.

------------- Database
UserRepository - хранение User. Примитивные функции сохранение,
 удаление, получение, обновление (CRUD).
