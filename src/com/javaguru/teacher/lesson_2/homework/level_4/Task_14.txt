Найти в интеренете ответ на вопрос:
Как запустить откомпилировнный код с помощью JDK и виртуальной
машины Java (JVM), без помощи среды разработки (из консоли)?



javac - заходим в папку проектаб находим файл с расширением .java, открываем cmd.exe , через /cd даем путь до него
прописаваем javac Main.java ( создается class Main )
java - java Main  - запускаем компиляцию
